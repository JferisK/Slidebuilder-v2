# SlideForge

PowerPoint Slide Drafting Tool mit GitHub Copilot Integration.

## Start

```
npm install
npm run dev
```

→ http://localhost:5173

## Verwendung

1. PPTX-Vorlage hochladen (Drag & Drop oder Datei-Dialog)
2. Folienmaster und Layout auswählen
3. Inhalte im rechten Panel bearbeiten
4. Klick auf Slide → Kommentar schreiben → Prompt in Copilot Chat einfügen
5. Als PNG exportieren

## Folienmaster hinzufügen

Einfach eine neue PPTX hochladen — alle Master und Layouts werden automatisch
erkannt.
